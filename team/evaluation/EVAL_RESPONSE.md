# Eval Response

1. Ball flew into space / lag
    The ball flying into space was a minor issue with the map layout since no colliders were placed at the corners. The issue has since been fixed. The lag was due to incapability of the user's computer, which involves little input on our end because it does not match our expected minimum hardware requirements.

2. Customization
    Special skins for gameplay elements were requested. It is now in-progress, near completion. 

3. Name uniqueness
    Out of scope for this project since it has a small userbase & deadline requirements. 

4. Controls
    Arrow keys instead of WASD; in scope & planned in addition to WASD.

5. Paddle speed
    Faster paddle with respect to ball is a simple & easy change we will do.

6. Speed indication
    In progress with a better rally system.

7. Adding back button to profile page
    Done


Overall, the general feedback is to polish the game up. The game required lots of complex parts, and so we now have to clean up some smaller parts that we have not prioritized until now.
